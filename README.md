# Revanth Kumar | AI & Data Engineer Portfolio

This is a production-ready, dark-themed personal portfolio built to showcase my projects in Machine Learning, Computer Vision, and Data Analytics.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** GitHub Pages (Static Export)

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure
- `src/components`: Contains the core sections (`Hero`, `About`, `Skills`, `Projects`, `Experience`, `Content`).
- `src/app`: Contains the global layout and the main page.
- `public`: Static assets including resume.

## Deployment
This project is configured for static export to be hosted on GitHub Pages via GitHub Actions (`.github/workflows/deploy.yml`).
